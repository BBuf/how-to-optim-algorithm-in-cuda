# pytorch

PyTorch 相关技术博客翻译与代码实现。

## 目录结构

```
pytorch/
├── code/          # 代码实现（float8、triton_all_gather 等）
├── attention/     # Flash Attention、Flex Attention、GQA 相关
├── compile/       # torch.compile、Triton 编译、depyf
├── training/      # FSDP、Checkpoint、TP 训练优化
├── quantization/  # FP8、GPTQ、INT4、torchao 量化
├── inference/     # GPT Fast、LLM 推理加速
└── kernels/       # CUTLASS、TMA、GEMM kernel 深度解析
```
